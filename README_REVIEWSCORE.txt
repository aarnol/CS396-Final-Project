To download the data, go to https://www.kaggle.com/datasets/mohamedbakhet/amazon-books-reviews
Use pip to install review_pred_requirements.
data_cleaning.ipynb can be used to clean the dataset and put the results into a dir named cleaned_data
To train and test a classification model, run reviews.ipynb
If you don't have a cuda compatible GPU you may have to change the device to cpu
To analyze sentiment, run sentiment_analysis
